# خدمة كشف سرطان الجلد

هذا المجلد يحتوي على خدمة كشف سرطان الجلد باستخدام Flutter و Flask API.

## الملفات

### 1. `api_services_disease.dart`
الخدمة الرئيسية للتعامل مع API الخاص بكشف سرطان الجلد.

**الميزات:**
- إرسال الصور للتحليل
- استقبال النتائج مع مستوى الثقة
- ترجمة أسماء الأمراض للإنجليزية والعربية
- معالجة الأخطاء
- التحقق من حالة الخادم

**الاستخدام:**
```dart
final apiService = DiseaseDetectionApiService();
final result = await apiService.detectSkinDisease(imageFile: selectedImage);

if (result.isSuccessful) {
  print('نوع المرض: ${result.diseaseNameEnglish}');
  print('مستوى الثقة: ${result.confidencePercentage}%');
  print('الوصف: ${result.diseaseDescriptionEnglish}');
}
```

### 2. `disease_detection_screen.dart`
صفحة واجهة المستخدم لكشف سرطان الجلد.

**الميزات:**
- اختيار الصور من المعرض أو الكاميرا
- عرض الصورة المختارة
- تحليل الصورة وإظهار النتائج بالإنجليزية
- إمكانية نسخ النصوص
- زر للدخول على Gemini مع اسم المرض
- واجهة سهلة الاستخدام

### 3. `disease_detection_card.dart`
بطاقة للانتقال إلى صفحة كشف سرطان الجلد.

## أنواع الأمراض المدعومة

| الكود | الاسم الإنجليزي | الاسم العربي |
|-------|----------------|-------------|
| nv | Nevus (Mole) | شامة حميدة |
| mel | Melanoma | سرطان الجلد الميلانيني |
| bkl | Benign Keratosis-like lesions | آفات شبيهة بالتقرن الحميد |
| bcc | Basal Cell Carcinoma | سرطان الخلايا القاعدية |
| akiec | Actinic Keratosis | التقرن الشعاعي |
| vasc | Vascular lesion | آفة وعائية |
| df | Dermatofibroma | ورم ليفي جلدي |

## الميزات الجديدة

✅ **النتائج بالإنجليزية** - جميع النتائج تظهر باللغة الإنجليزية  
✅ **إمكانية النسخ** - يمكن نسخ أي نص بالضغط على أيقونة النسخ  
✅ **زر Gemini** - للدخول على Gemini مع اسم المرض لطلب العلاج  
✅ **واجهة محسنة** - تصميم أفضل وأكثر وضوحاً  

## متطلبات التشغيل

1. **Flask API**: يجب تشغيل خادم Flask على المنفذ 5000
2. **نموذج TensorFlow**: يجب وجود ملف `best_by_train_acc.h5`
3. **الصور**: يجب أن تكون الصور بصيغة JPG أو PNG
4. **url_launcher**: للتكامل مع Gemini

## كيفية الاستخدام

1. **من الصفحة الرئيسية**: اضغط على بطاقة "كشف سرطان الجلد"
2. **اختيار الصورة**: من المعرض أو الكاميرا
3. **تحليل الصورة**: اضغط على "تحليل الصورة"
4. **عرض النتائج**: ستظهر النتائج بالإنجليزية مع إمكانية النسخ
5. **استشارة Gemini**: اضغط على "Ask Gemini about treatments"
